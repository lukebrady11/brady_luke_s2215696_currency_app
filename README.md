Overview

This Android application retrieves British Pound (GBP) exchange rates from an RSS feed:

https://www.fx-exchange.com/gbp/rss.xml

It displays:

- A summary of three main currencies (USD, EUR, JPY)

- A full searchable list of all currencies

- Coloured strength ratings; Green, Light green, Yellow, and Red (Strong - Weak currencys)

- Country flag icons

- Tap-to-convert screen with two-way conversion

- Automatically updates every 15 mins

- Supports portrait and landscape layouts

Technologies Used

- Java

- Android SDK

- RecyclerView + Adapter

- XmlPullParser

- ExecutorService + ScheduledExecutorService

- Custom layouts for portrait & landscape
